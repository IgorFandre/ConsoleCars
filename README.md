# Cars Console App

Небольшое консольное приложение на C#.
Пользователь вводит марку автомобиля, программа выводит описание.

## Что лежит в файлах

- Program.cs: точка входа, цикл ввода, проверка команды "конец", вывод результата.
- Cars.cs: enum CarType, интерфейсы, абстрактные классы, классы машин и фабрика CarFactory.

## Машины на выбор

- Tesla
- Toyota
- Lada
- Geely
